b tag

The b tag is used to draw the users attention to the element's. This was formaly known as boldface element, and most browser still draw the text in boldface.

i tag

The i tag is used to make the element style in italic formate.

u tag

The u tag is used to underline the element.

select tag

The select tag allows you to create a selection tool in which you can have several options to choose from.

button tag

The button tag can be used to create a button and can be activated by a user with mouse, keyboard, finger, voice command. Once activated, it then performs a programmable action, such as submiting a form or opening a dialog.

hr tag

The hr tag will create a line or a

break between paragraph for example - a shift of topic within a section

br tag

The br tag is used to break
the line mostly used for writing poem or an address, where the division of lines is significant.
